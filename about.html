About This Lab

This Cybersecurity Password Lab is an educational web-app designed to help users understand password strength, common weaknesses, and how modern attackers guess passwords.

The analyzer uses AI-style JavaScript heuristics to detect weak patterns such as:

  • Short passwords with low entropy
  • Repeated characters (e.g., aaaaaa)
  • Sequential patterns like 123456 or qwerty
  • Keyboard-based patterns and dictionary words
  • Simple substitutions (e.g., P@ssw0rd)
